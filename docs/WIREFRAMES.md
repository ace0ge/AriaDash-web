## 设计原则

1. **拇指操作区** — 所有操作按钮放在屏幕下半部分，便于单手操作
2. **信息密度适中** — 每个任务卡片展示 3-4 行关键信息，不堆砌
3. **视觉层级** — 文件名 > 进度 > 速度/ETA
4. **卡片无按钮** — 避免误触，操作通过手势/详情页/多选模式完成
5. **反馈即时** — 所有操作立即 UI 反馈，不等待网络响应
6. **手势一致性** — 左滑暂停/恢复，右滑删除，主页卡片与 iOS 原生返回手势同向不冲突
